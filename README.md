# LiveResume - 可视化数据驱动简历生成器

![Vue3](https://img.shields.io/badge/Vue-3.x-4FC08D?style=flat-square&logo=vue.js)
![Element Plus](https://img.shields.io/badge/Element_Plus-2.x-409EFF?style=flat-square&logo=element)
![ECharts](https://img.shields.io/badge/ECharts-5.x-AA344D?style=flat-square&logo=apache-echarts)

> 一个基于 Vue 3 Composition API 构建的响应式简历生成系统，结合数学建模数据可视化

## 🛠 技术架构 (Tech Stack)

- **核心框架**: Vue 3 (Composition API) - 利用响应式系统实现数据与视图的实时双向绑定。
- **UI 组件库**: Element Plus - 构建高效的表单交互与配置面板。
- **数据可视化**: Apache ECharts 5 - 建立多维能力雷达图，量化技能掌握度。
- **样式引擎**: Tailwind CSS - 原子化 CSS 实现快速布局与移动端适配。
- **排版算法**: CSS Print Media Query - 解决 Web 端像素到 A4 纸张毫米级的精确映射，支持自动分页与内容防截断。

## 🚀 核心特性

1. **实时渲染引擎**: 左侧编辑数据，右侧毫秒级动态预览。
2. **可视化数据建模**: 将技能点转化为直观的雷达图与进度条。
3. **智能分页算法**: 自动计算内容高度，动态模拟 A4 纸张分页效果。
4. **纯前端架构**: 无需后端，单文件组件 (SFC) 思想实现轻量化部署。
